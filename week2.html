Week 2 – Structuur, classes en dupliceren

Wat heb ik deze week gedaan?

  • De eerste HTML-pagina gedupliceerd en hernoemd naar week2.html.
  • Op elke pagina dezelfde navigatie geplaatst.
  • Begonnen met het gebruiken van een id (id="hoofdmenu") voor het menu.
  • Voorbereiding gemaakt voor CSS (container, marges, typografie) maar nog niet toegepast, omdat dit pas later mag.

Waarom is dit belangrijk?

  • Als elke pagina dezelfde navigatie heeft, kan de leerkracht makkelijk doorklikken.
  • Door nu al een structuur te maken, wordt CSS later veel eenvoudiger.
  • Het lijkt meer op een “echte” website met meerdere pagina’s.

Wat komt later (volgens de opdracht)?

In de les staat dat we later een styles.css moeten maken met:

  • een achtergrondkleur
  • een font van Google Fonts (bv. Roboto)
  • een container met marges (zoals in het voorbeeld)

Ik heb dit nog niet toegevoegd omdat de opdracht nu zegt: “momenteel uitsluitend tekst in HTML”.
